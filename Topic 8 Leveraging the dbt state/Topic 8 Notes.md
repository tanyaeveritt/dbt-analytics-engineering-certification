## Using dbt retry
dbt retry re-executes the last dbt command from the node point of failure.
Retry works with the following commands: build, compile, clone, docs generate, seed, snapshot, test, run, run-operation

[Learn more abotu retry - dbt documentation](https://docs.getdbt.com/reference/commands/retry)
