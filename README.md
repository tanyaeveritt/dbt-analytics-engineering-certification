## ðŸ“ Study Tips for Gen AI Specialty

*   **Defer, State, and Clone:** Understand the differences between these concepts. These came up so often.
*   **Test vs. Model Contracts:** Know the difference between test contracts and model contracts, and understand the difference between generic and singular tests.
*   **Tests, generic, custom generic** how its written and referenced, yml, sql
*   **Upstream and Downstream:** Familiarize yourself with the flow of data from source to staging and then into the marts, with multiple models feeding into the marts. Union, joins
*   **dbt build --fail-fast** - Mutiple times mentioned
*   **--indirect-selection** - I didn't event study this but it came up
*   **Troubleshooting and Error Resolution:** Practice diagnosing errors and understanding the structure of your files.
*   **Exam Format:** Get used to dropdown lists instead of fill-in-the-blanks, and practice drag-and-drop questions for ordering steps. No heatmaps or type in for me.
*   **Limitations of python model**
*   **Git and GitHub:** Be comfortable with Git commands and know how to create and submit a pull request. These were the ordering steps
*   **Exposure** how its written and referenced
*   **Using jinja** in model .sql file itself e.g. using jinja to create a new column based on condition
